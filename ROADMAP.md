# Roadmap

## Current focus

- production trust for FastAPI teams
- accurate docs and examples
- stronger permission and scoping behavior
- stable CI and release hygiene

## Next milestones

- richer detail pages and audit timelines
- better dashboard customization
- improved neutral theme support alongside the Matrix theme
- more polished example applications and migration guides

## Non-goals for now

- broad multi-ORM parity
- a separate hosted control plane
